# maCher Chan 字体规范文档

## 字体文件
- 文件名：HelveticaNeue93BlackExt（位于 assets/fonts/）
- 格式：OTF 或 TTF
- CSS @font-face 声明名称：'HelveticaNeue93BlackExt'

---

## 使用场景与参数

### 1. 品牌名 — maCher Chan
- 字体：HelveticaNeue93BlackExt
- 大小写：严格保留原始大小写 "maCher Chan"（m小写, a小写, C大写, h小写, e小写, r小写, 空格, C大写, h小写, a小写, n小写）
- 字距：letter-spacing: -0.077em（对应 -77）
- 使用位置：首页大标题、导航栏左侧品牌名、所有页面出现品牌名的地方
- 注意：任何情况下不得将品牌名改为全大写或全小写

### 2. 副标题 — DESIGN & TREND STUDIO
- 字体：HelveticaNeue93BlackExt
- 大小写：全大写
- 字距：letter-spacing: -0.077em（对应 -77）
- 使用位置：首页右下方

### 3. 导航栏链接 — PROJECTS / STUDIO / SERVICES / CONTACT
- 字体：HelveticaNeue93BlackExt
- 大小写：全大写（text-transform: uppercase）
- 字距：letter-spacing: -0.023em（对应 -23）
- 字号：16px
- 使用位置：所有页面的顶部导航栏

### 4. 页面英文标题 — 如 SELECTED WORKS、OUR SYSTEM、GET IN TOUCH 等
- 字体：HelveticaNeue93BlackExt
- 大小写：全大写
- 字距：letter-spacing: -0.023em（对应 -23）

### 5. 中文标题 — 如 案例展示、静奢主义、服务套餐 等
- 字体：微软雅黑 Bold（系统字体）
- CSS：font-family: "Microsoft YaHei", sans-serif; font-weight: bold;

### 6. 中文正文
- 字体：微软雅黑 Regular
- CSS：font-family: "Microsoft YaHei", sans-serif; font-weight: normal;
- 字号：14-16px
- 行高：1.8

### 7. 英文正文 / 标签 / 小字
- 字体：HelveticaNeue93BlackExt
- 大小写：视内容决定（标签全大写，正文句首大写）
- 字距：letter-spacing: -0.023em（对应 -23）
- 字号：12-14px

---

## CSS @font-face 声明

```css
@font-face {
  font-family: 'HelveticaNeue93BlackExt';
  src: url('../assets/fonts/HelveticaNeue93BlackExt.otf') format('opentype');
  /* 如果是ttf格式则改为 format('truetype') */
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```

## CSS 变量

```css
:root {
  --font-brand: 'HelveticaNeue93BlackExt', 'Helvetica Neue', Arial, sans-serif;
  --font-cn: 'Microsoft YaHei', sans-serif;
  
  --tracking-brand: -0.077em;    /* -77 品牌名/大标题 */
  --tracking-nav: -0.023em;      /* -23 导航/小标题 */
}
```

## 禁止事项
- 禁止在任何位置使用品牌字体以外的英文字体（如 Arial、Roboto、系统默认无衬线）
- 禁止将 "maCher Chan" 写成 "MACHER CHAN" 或 "macher chan"
- **唯一例外**：首页大标题允许全大写 "MACHER CHAN"，因超大字号下全大写视觉更和谐
- 导航栏品牌名、页脚、其他页面出现的品牌名一律保持混合大小写 "maCher Chan"
- 禁止使用 letter-spacing: 0 或正值字距
- 禁止中文内容使用品牌英文字体
